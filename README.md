# 工程數學期末報告：傅立葉級數與 RLC 電路分析

本報告針對期末考題進行深入的數學建模與物理意義分析，主要探討**方波信號的傅立葉級數展開（訊號完整性分析）** 以及 **RLC 串聯濾波器的選頻特性**。

---

## 📝 題目一：方波信號分析（時鐘信號與傳輸線效應）

### 1. 問題描述
數位電路中，時鐘信號是標準的週期性方波。某 FPGA 系統的時鐘週期為 $T=10\text{ ns}$（對應頻率 $100\text{ MHz}$），高電平 $V_{OH}=3.3\text{ V}$，低電平 $V_{OL}=0\text{ V}$，占空比為 $50\%$。本題目要求將該時鐘方波展開為傅立葉級數，分析其頻率成分，並解釋方波時鐘經長導線傳輸後邊沿變緩的原因。


## 📝 題目二：RLC 電路頻率回應（選頻特性分析）

### 1. 問題描述
RLC 串聯帶通濾波器是射頻接收機的核心選頻電路。已知參數：$R=10\ \Omega$, $L=1\text{ mH}$, $C=2.53 \times 10^{-8}\text{ F}$。輸入為包含直流分量、基波、3次諧波的週期矩形脈衝信號，基波頻率 $f_0=1\text{ MHz}$（週期 $T=1\ \mu\text{s}$）。輸出為電容兩端電壓 $v_o(t)$，要求用傅立葉級數分析濾波器的選頻特性。
